📖 项目简介

Legado（开源阅读） 是一款广受欢迎的开源阅读应用，主打 高度可定制 + 书源自由。

它不提供任何内置内容，而是通过「书源规则」的方式，让用户自行接入全网小说、文学站点与内容源，从而构建一个完全属于自己的数字图书馆。

✨ 核心特性一览

  • 📚 多书源聚合
    支持自定义书源规则，多个站点自动聚合、去重、择优

  • 🎨 高度界面定制
    字体、行距、段距、背景、翻页动画，想怎么调就怎么调

  • ☁️ 跨设备同步
    支持 WebDAV，同步书架、阅读进度、书签与设置

  • 🔊 TTS 听书模式
    内置多种语音引擎，通勤、摸鱼两不误

  • 🛡️ 纯净阅读体验
    无广告、无推荐流、无信息投喂，只有书

  • 🧩 可扩展规则体系
    书源 / 替换规则 / 净化规则，进阶用户的快乐老家

🚀 快速上手指南

1️⃣ 导入书源（第一步也是最关键的一步）

Legado 本身不提供内容，书源决定一切。

  • 推荐书源合集：
    👉 https://www.yckceo.com/yuedu/shuyuan/index.html

  • 打开方式：

    • 书源管理 → 网络导入
    • 粘贴链接 → 确认导入

新手建议先用“精选书源”，不要一口气全导。
